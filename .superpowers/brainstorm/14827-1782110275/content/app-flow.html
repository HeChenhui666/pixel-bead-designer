A · 线性向导流

每步一个屏幕，顺序推进。上传 → 设置 → 编辑 → 导出，步骤清晰，不走回头路。适合第一次使用的用户。

优点

  • 流程清晰无歧义
  • 实现简单

缺点

  • 来回跳转不方便
  • 编辑时不能看导出

B · 编辑器为核心 + 底部 Tab

主页直接就是图纸编辑器，通过底部 Tab 在图片、编辑、色卡、历史间快速切换。编辑体验沉浸，适合反复使用的用户。

优点

  • 编辑体验流畅
  • 功能随时可达

缺点

  • 新用户稍有学习成本
  • 小程序适配需关注

C · 历史列表 + 全屏编辑器

首页是作品列表（含新建入口），点击后进入全屏编辑模式。历史管理和编辑分离，对多项目用户友好，类似相册应用体验。

优点

  • 多项目管理直观
  • 编辑空间最大化

缺点

  • 层级多一层
  • 单次使用略啰嗦
